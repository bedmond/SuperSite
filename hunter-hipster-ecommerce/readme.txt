CSS
----------

The theme stylesheet is css/style.default.css. If you want to make any changes, you can do it here or better to override it in custom.css so you can update the original theme stylesheet if an updated is released.  

There are 6 other colour variant stylesheets prepared. You can switch to any colour variant just by replacing css/style.default.css to e.g. css/style.red.css in the <link> element in the <head>s.

LESS
----------
The easies way to change i.e. theme colours, fonts and any CSS rules in general is by editing the LESS files. In the LESS directory, there is a main file style.default.less. It takes control of all the necessary module includes. The modules (blog, footer, navbar, pages, etc.) are located in the modules subdirectory. Most modifications can be done by configuring the theme in variables-custom.less.

Javascript
----------

Apart from Bootstrap JS components majority of JS is located in /js/front.js. If you want to make any js settings finetuning, you can do it here.
